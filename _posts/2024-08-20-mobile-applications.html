During my Mobile Sensing and Learning class, I had the opportunity to create a mobile app of my choosing for the final project. What I created was a fitness based game called Monster Trainer, where users would level up their characters (monsters) by meeting the fitness goals set for them in the app. This project has given me valuabel skills in Mobile User Interface Design, Model View Controller (MVC) architecture, as well as Problem Solving and Creativity.

Overview

The app has multiple functions that the user can utilize. The features are as follows:

  • The user can select their monster that they would like to train
  • The goals set by th app are visible from the home page, as well as the mosnter they have chosen
  • They can view the stats of their monster as well as a brief description of it by tapping on it
  • Fitness statistics such as the amount of stairs they climbed, steps they have taken, etc are displayed in the fitness tracking section
  • The user can see all of the monsters in the game, as well as their stats and a brief description
  • Lastly, the user can battle AI opponents with their chosen monster
